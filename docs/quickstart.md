# Quickstart

## Installation

```bash
git clone https://github.com/vxssroot/web6-10g-lab.git
cd web6-10g-lab
make install
```

## Basic Commands

```bash
# Check system status
10g status

# Run network simulation
10g simulate

# View topology
10g topology

# Run full example
python examples/basic_simulation.py
```

See `simulator/README.md` for simulation details.
