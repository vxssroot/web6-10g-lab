# Simulation Layer

This module provides high-fidelity discrete-event simulation capabilities for the 10G autonomous networking stack. It models node behavior, AgentNet consensus, QuantumLink channels, and MeshCompute resource dynamics under various conditions.

## Features
- Node lifecycle management
- Latency and packet loss modeling
- AgentNet consensus plane simulation
- QuantumLink entanglement and key distribution simulation
- Real-time telemetry generation

## Usage
```bash
python -m simulator.network_sim
```
